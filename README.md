# Learning OpenAi-Gym Boxing

This is the project for generating the learning agent of Atari 'Boxing game' using reinforcement learning and OpenAi-Gym.

## Getting Started

### Environment

Pip modules

```
Python == 2.7.13

Gym == 0.9.3

Tensorflow == 1.0.1
```

## Running the Program

```
$ python main.py
```
